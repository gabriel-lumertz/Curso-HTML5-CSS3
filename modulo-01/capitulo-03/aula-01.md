# A diferença entre HTML, CSS e JavaScript

* HTML
    * **H**YPER**T**EXT **M**ARKUP **L**ANGUAGE
        * Linguagem focada em **conteúdo**
            * Texto
            * Imagens
            * Vídeos
            * Tabelas


* CSS
    * **C**ASCADING **S**TYLE **S**HEETS
        * Linguagem focada em **design**
            * Cores
            * Sombras
            * Tamanhos
            * Posicionamento

* JavaScript
    * Linguagem de programação
        * focada em interações
            * Menus
            * Animações
            * Popups
            * Validações
